# 🤖 Agent-chat-ui

This module provides a modern graphical user interface (GUI) to interact with the **Email Assistant** using the Vercel AI infrastructure.

## 🚀 Quick Start Guide

Follow these steps to connect your LangGraph agent with the web interface:

### 1. Start the Agent Server
Open your terminal inside the project folder (where your LangGraph configuration is located) and run the development environment:

```bash
uv run langgraph dev
```

### 2. Configure the Web Interface
Once the server is running:

1. Go to [agentchat.vercel.app](https://agentchat.vercel.app).
2. Upon entering, you will see a configuration form.
3. **Do not change any values** (Deployment URL, Assistant/Graph ID, LangSmith API Key, etc.).
4. Click **Continue**.

### 3. Ready to go
Your agent is now live in the browser! 🚀
